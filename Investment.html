成本

1、动态资金成本 = 静态资金成本 + 利息 + 流动资金

2、静态资金成本 = 前期开发成本 + 项目建设成本 + 建设期土地成本 + 其他成本，其中：

    前期开发成本 = 项目前期含税开发费用之和

    项目建设成本 = （设备成本 + 工程成本 + 服务成本 ） * 税率

    土地成本 = 土地招拍成本 或 建设期间的土地租赁成本

    其他成本 = 青苗补偿成本 + 乡村基金振兴 + 产业发展基金 + 居间费用

 

收入

1、项目净收入 = 发电量 * 有效发电时长 * ( 1 - 销项税 ) ，其中：

    发电量 = 装机规模 * 有效发电时长 * 弃光量占比

        装机规模 = 最小发电单元理论发电功率 * 最小发电单元的数量

        有效发电时长 = 当地理论发电时长 * 发电效率 * 历年衰减值 * 背面增益效率

    电价 = 上网电价 或 市场化电价

    销项税 = 项目总收入 * 销项税率
